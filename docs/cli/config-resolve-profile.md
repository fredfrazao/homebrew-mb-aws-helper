# config resolve-profile

Resolve the final AWS profile for environment and service.

## Command

```bash
mb-aws-helper config resolve-profile --help
```

## Help output

```text
usage: mb-aws-helper config resolve-profile [-h] env {artifactory,gitlab}

positional arguments:
  env                   prod | sandbox | int
  {artifactory,gitlab}  Service name

options:
  -h, --help            show this help message and exit
```

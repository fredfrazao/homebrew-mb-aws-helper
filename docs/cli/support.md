# support

Artifactory support bundle helpers.

## Command

```bash
mb-aws-helper support --help
```

## Help output

```text
usage: mb-aws-helper support [-h] {instances,status} ...

positional arguments:
  {instances,status}
    instances         Run jfrog_support_collect.sh via SSM
    status            Show status of an SSM CommandId

options:
  -h, --help          show this help message and exit
```

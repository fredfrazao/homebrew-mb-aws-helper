# config profiles

Show configured AWS profile patterns.

## Command

```bash
mb-aws-helper config profiles --help
```

## Help output

```text
usage: mb-aws-helper config profiles [-h] [--json] [name]

positional arguments:
  name        Optional profile name to show only one profile

options:
  -h, --help  show this help message and exit
  --json      Output JSON
```
